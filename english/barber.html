Barbershop Landing Page

HTML SASS JavaScript

Details

This project was developed with the goal of creating a functional and attractive website that reflects the essence of the barbershop and facilitates client contact. The creation process included the following stages:

  1. Planning and Structuring: Identifying the barbershop's needs, defining key objectives, and mapping the customer journey to create a digital experience aligned with the space's concept.
  2. Frontend Development: Building a modern and responsive interface using HTML, SCSS, JavaScript, and APIs, ensuring the website is visually appealing, functional, and accessible on both mobile and desktop devices.
  3. Backend Configuration: Implementing interactive elements to allow users to navigate and interact with the site, including navigation buttons, a map with the space's location, and service prices.
  4. Optimization and Testing: Conducting rigorous usability and functionality tests to ensure smooth and responsive navigation, aligned with client expectations.

The result is a website that represents the barbershop's identity, facilitates service scheduling, and promotes a connection between the brand and its clients, standing out as an efficient and professional communication channel.
